# ThevaaramSongs
Personalized Music App for Music Teachers. Having the app with own playlist for their students

* Jetpack Compose for UI
* Material3

* Media3 ExoPlayer

* Compose Glide
* Dagger Hilt
* Coroutines
* Flow
* Bitrise and Github Actions for C
